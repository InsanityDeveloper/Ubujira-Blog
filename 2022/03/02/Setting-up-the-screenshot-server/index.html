I wanted to add another useful service to my homserver, which i decided was going to be a Screenshot-Server.

Something i prioritized was that it had ShareX integration and a dashboard.

On GitHub, I found XBackBone, something which seemed to be perfect for this.

Installation

I made a new directory inside /var/www/html/cdn.ubujira.dev, and then downloaded the latest release of XBackBone with wget https://github.com/SergiX44/XBackBone/releases/download/3.5.1/release-v3.5.1.zip.

Then unzip the zip file with unzip release-v3.5.1.zip.

After this setup a new nginx site with this as the basis of it.

After doing that visit https://DOMAIN.TLD/install and install it.
