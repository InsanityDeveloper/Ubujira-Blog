I wanted to start writing a blog as part of my Homeserver, so I had to find a suitable blogging system, which i can implement easily and start writing.

I with some googling found Hexo and it seemed pretty much perfect for the task.

Installation

To install Hexo, you first require NPM to be installed.
After that you can install Hexo with the command npm install hexo.

Then use the command hexo init (Directory) to initialize a blog.

After that youâ€™ll want to edit the _config.yml and edit the values to how you want them. Configuration options.

After that I made it integrate with Git and with Git made it automatically serve the static pages on my nginx webserver.

Adding an admin panel

For the admin panel, I found hexo-admin.

To install it, run npm install --save hexo-admin.

Then to access the admin panel run hexo server -d.
Make sure to setup a password on at the minimum on the Server itself and if possible, do it on the nginx site itself on a seperate domain using a reverse proxy.
